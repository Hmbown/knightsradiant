# Knights Radiant

A Hermes skill pack for rotating engineering work through distinct lenses instead of flattening every task into one generic persona.

Best for:
- structured code review and refactor passes
- choosing the right engineering lens before acting
- preserving hand-off artifacts between architecture, implementation, observability, reliability, and rollout stages

Starter skills:
- `choose-order` — select the right order for the current task
- `run-cycle` — choose an appropriate cycle and hand-off plan
- then one active order skill at a time: `elsecaller`, `lightweaver`, `truthwatcher`, `edgedancer`, `dustbringer`, `skybreaker`, `windrunner`, `willshaper`, `stoneward`, `bondsmith`

Shelf rule:
Only load one active order at a time unless you are intentionally routing or planning the sequence. The artifact, not the theme, is what carries the work forward.
