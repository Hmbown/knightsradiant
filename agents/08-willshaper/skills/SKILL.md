# Willshaper Skills

Willshaper skills make the repo easier to build in and harder to misuse. Use them when conventions should become structure, scripts, or scaffolds instead of social rules.

## Available skills

- [`scaffold-rite.md`](./scaffold-rite.md) — create a new package, service, job, or CLI that matches repo conventions.
- [`build-reshape.md`](./build-reshape.md) — simplify build and CI structure for speed, consistency, and clarity.

## Usage contract

1. Start from existing conventions or create them deliberately; do not improvise per module.
2. Keep templates honest: minimal, owned, and readable.
3. Measure build changes with clean-checkout and CI reality, not local hunches.
4. Escalate to Elsecaller when structure work reveals a deeper architectural mismatch.
