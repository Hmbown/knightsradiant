# Elsecaller Skills

Elsecaller skills make structural decisions explicit before the repo hardens around them. Use them when the costliest part of the task is choosing the shape, not typing the code.

## Available skills

- [`adr-foresight.md`](./adr-foresight.md) — produce an ADR with real alternatives, tradeoffs, and migration thinking.
- [`system-cartography.md`](./system-cartography.md) — map the current system's boundaries, dependencies, and main flows.

## Usage contract

1. Name constraints and non-goals before drawing a target shape.
2. Compare viable alternatives; do not present one option as destiny.
3. Record what is reversible and what is not.
4. Escalate to Lightweaver when the problem is still conceptually or linguistically muddy.
